HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements can be divided into smeantic and non-semantic.
  • src and alt attributes are important wehen inserting an image.
  • src attribute defines the location of the image.
  • alt attribute contains a string of text that gives a description and functionality of the inserted image.
  • Anchor element creates a link to a webpage and it is done using href attribute which point to the URL for the link.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are thre types of CSS: inline, internal, and external. The most common type is external CSS
  • Inline CSS is the least flexible because they have to be editted individually.
  • Internal CSS: entire stylesheet is embedded using style element within the head element.
  • External CSS: We link the CSS file using a link element in the html head element.
  • Declaration block is specific to CSS and it consists of a vlaue and a property.
